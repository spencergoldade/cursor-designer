# Lean rules profile (teaching/optional)

- Minimal set of rules for workshops or very small projects.
- This document is a **quick, implementation-focused companion** to the profiles section in `README.md`.

## Which files to copy for the lean profile

Copy only these into `.cursor/rules` if you want the lightest setup:

- `core/design-core.mdc`
- `core/cursor-behavior-constraints.mdc`
- `frontend/ui-layout-and-density.mdc`
- `frontend/ux-forms-and-validation.mdc`
- `frontend/accessibility-frontend.mdc`
- `frontend/ux-flows-and-feedback.mdc`

You can later add more rules from `frontend/` and `binders/` as your needs grow.

For details on **core vs lean vs full design-ops profiles**, see the "Profiles: core, lean, and full" section in `README.md`.
